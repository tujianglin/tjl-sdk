# tjl-sdk

工具组件库基础模板

## 基础使用
``` javascript
// 使用方法
import { add } from 'tjl-sdk'
add(1, 2)
```


## API 目录

###  普通方法

- [add](#add)  两数字相加

###  文档方法

- [previewFile](#previewFile)  预览文本(可扩展其他预览)

## API 说明

### previewFile
              
 预览文本(可扩展其他预览)

```javascript
* previewFile({url: 'http://xxx.xxx/xx.txt', type: 'txt'})
```

### add
              
 两数字相加

```javascript
* add(1, 2) => 3
```
